# Contributing to 4rm0Byte 🚀

Thank you for your interest in **4rm0Byte**! 🎉

## Contributions Not Accepted 🚫

At this time, **4rm0Byte** is a personal project and we are not accepting contributions. This repository is maintained solely for personal use and updates related to my blogging website.

However, I may consider **exceptional, high-value contributions** on a case-by-case basis.

### Why No Contributions?

- **Personal Project**: This project is intended for personal use and customizations.
- **Limited Scope**: I am managing all aspects of this project myself.
- **Privacy and Security**: As a personal project, I prefer to keep control over all aspects of the code and content to maintain privacy and security.

### Exceptional Contributions (Case-by-Case)

A contribution is more likely to be considered if it is:

- A clear security fix
- A critical bug fix that affects site stability or correctness
- A meaningful performance improvement with measurable impact
- A documentation fix that removes serious confusion
- A small, focused change that aligns with project direction

## Security Issues

If you discover a security issue, please do **not** open a public issue first.
Report it privately via email: [Armoghan Blogs](mailto:armoghanblogs@gmail.com)
